    Thumb Helper.

    This helper allows you to generate thumbnails.

    package Default

     Methods

    Magic method. It dynamically calls all other methods.

    __call(string $name, array $params) : string

    Each method takes these arguments:

    • $path Path of the image from which to create the thumbnail. It can be a relative path (to APP/webroot/img), a full path or a remote url;
    • $params Parameters for creating the thumbnail;
    • $options Array of HTML attributes for the img element.
    see https://github.com/mirko-pagliai/cakephp-thumber/wiki/How-to-use-the-helper

    Resizes the boundaries of the current image to given width and height. An anchor can be defined to determine from what point of the image the resizing is going to happen. Set the mode to relative to add or subtract the given width or height to the actual image dimensions. You can also pass a background color for the emerging area of the image

    resizeCanvas() : 
    magic
    method Resizes the boundaries of the current image to given width and height. An anchor can be defined to determine from what point of the image the resizing is going to happen. Set the mode to relative to add or subtract the given width or height to the actual image dimensions. You can also pass a background color for the emerging area of the image

    Checks is a method name is an "Url" method.

    isUrlMethod(string $name) : boolean

    This means that the last characters of the method name are "Url".

    Example: cropUrl is an "Url" method. crop is not.
